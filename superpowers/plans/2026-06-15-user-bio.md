# User Bio Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a single `bio: string` field (0–200 chars) to user profiles. Editable from Edit Profile; visible to other users on the public profile when non-empty.

**Architecture:** One top-level field on `Profile` (no nested object, no new collection). Backend validates + trims; frontend hides the public bio when empty. Legacy profiles tolerate the missing field via defensive read.

**Tech Stack:** React Native + Expo Router (frontend) · Firebase Cloud Functions + Zod + Firestore (backend) · Jest (tests)
